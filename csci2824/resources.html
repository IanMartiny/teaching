LaTeX resources

LaTeX is a "programming language", more of a formatting language used to create documents (such as pdfs).

LaTeX is a widely used language throughout mathematics and computer science. If you continue working in computer science you WILL eventually need to learn LaTeX. May as well be now

There are many resources online that can be used to learn how to write LaTeX code. I've linked some below, you can also find plenty of others on Google

  • ShareLaTeX's beginners tutorial, explanations with videos
  • TeX User Group's (TUG's) tutorial pdf
  • A website literally called LaTeX Tutorial
  • Another professor here at CU has done a tutorial on LaTeX
